Back To Labels...

This will help you to setup automatic dates on labels

Step One: Go to the labels folder and select the label you want to edit
Step Two: Then open the label that you want to edit using P-touch
(Just double click as it should be installed)
Step Three: P-touch should open and load
Step Four: Edit the text where you want the date, for example after the "Use By"
(Remember to put a space)
Step Five: Then type "{DATE+X}" where X is the number of days forward from today
(Max. 9 Days Ahead)

Below, we show the results if the date is Sep 12

Code Result
{DATE} NO RESULT
{DATE+1} Sep 13
{DATE+2} Sep 14
{DATE+7} Sep 19
{DATE+10} NO RESULT
Step Six: Then press save and close P-touch
